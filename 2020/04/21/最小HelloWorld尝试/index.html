C/C++

实验环境：Ubuntu、GCC7.5.0
gcc编译选项：-fno-stack-protector -z execstack -z norelro -fno-builtin -O0，关闭常见的防护，关闭内置函数优化,关闭优化，动态链接，生成64位可执行程序

最常见的写法

先写一个最为常见的HelloWorld

#include<stdio.h>
int main()
{
    printf("Hello,World\n");
    return 1;
}

编译最后大小为6952字节

使用init段

#include<stdio.h>

static __attribute__((constructor)) void before_main(void) 
{
    printf("Hello,World\n");
}

int main()
{}

编译之后大小为7072字节，比常见的写法还大

使用全局变量

#include<stdio.h>

char tmp=printf("Hello,World\n");

int main()
{}

编译之后是7272字节，更大。与上面一种方法一样，只是想到了在main函数之前执行代码的方式，实现一下
